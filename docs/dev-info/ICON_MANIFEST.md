# Frank Energie Onbalansmarkt - Icon Manifest

This document lists all SVG icons created for the Frank Energie smart home app.

## Main App Icon

**File**: `/assets/icon.svg`
**ViewBox**: 500x500px
**Description**: Modern smart battery trading app icon featuring:
- Blue gradient battery with charge indicators
- Green upward trending arrows (energy trading profits)
- Orange euro symbol (financial transactions)
- Network connection dots (platform integration)
- Professional flat design with subtle depth

**Color Scheme**:
- Primary Blue: #00A8E8, #0066CC
- Energy Green: #00D084, #00A854
- Accent Orange: #FFB703, #FB5607

## Design Guidelines

### Color Palette

**Primary Colors**:
- Blue: #00A8E8 (light), #0066CC (dark)
- Green: #00D084 (bright), #00A854 (dark)
- Purple: #6C5CE7
- Orange: #FFB703 (light), #FB5607 (dark)
- Red: #FF6B6B (usage/cost)
- Gray: #666666 (neutral)

**Background Colors**:
- Light Blue: #F0F4FF
- Light Green: #F0FFF4
- Light Orange: #FFFAF0
- Light Red: #FFF0F0
- Light Gray: #F0F0F0
- Light Purple: #F5F0FF

### Style Characteristics

- **Stroke Width**: 1.5-2.5px for main elements
- **Border Radius**: 2-3px for rectangles, rounded joins on paths
- **Opacity**: Varied use of opacity (0.1-0.6) for depth and emphasis
- **Gradients**: Linear gradients for battery fills and transitions
- **Icons**: Simple, recognizable at small sizes (24x24px minimum)
- **Scalability**: 200x200px viewBox allows scaling to any size
- **Clarity**: Minimal detail, maximum readability

### Usage

These icons are designed for use in:
- Homey app capability displays
- Device settings and configuration UI
- Flow cards and automation triggers
- Mobile and web dashboards
- Light and dark themed interfaces

All icons work well on both light and dark backgrounds due to the color-coded design approach.
